Tema: 2.8 Funcion inversa.

Objetivo:Es comprender como encontrar la funcion inversa de una funcion dada y como utilizarla para resolver problemas relacionados con el comportamiento de la funcion original. La funcion inversa es importante en calculo diferencial ya que permite encontrar la derivada de una funcion inversa. Esto es util para resolver problemas de optimizacion y para encontrar la tasa de cambio inversa de una funcion.

Informacion general

La funcion inversa es fundamental para el desarrollo de calculo integral, ya que es una herramienta importante para resolver problemas de integracion y para el cambio de variables en integrales definidas, con el fin de resolver problemas relacionados con el comportamiento de la funcion original, la optimizacion, la tasa de cambio inversa y la integracion.

Definicion del tema

El tema de funciones inversas proporciona una comprension fundamental de como las funciones se relacionan entre si y como se pueden utilizar para resolver problemas en una variedad de campos, incluyendo las matematicas, la fisica, las ciencias sociales y la programacion.

Funciones inversas

Una funcion es una relacion entre dos variables, de manera que para cada valor de la variable independiente existe a lo mas un unico valor asignado a la variable independiente por la funcion. Imagina que tienes la funcion y = f(x). Tu le das un valor (x) y ella te devuelve otro (f(x)). Una buena idea seria encontrar una funcion que cuando le demos el valor f(x) nos devolviera x, es decir, una maquina que haga la transformacion inversa de f(x). En otras palabras, queremos encontrar una funcion que deshace la transformacion que ocasiona la funcion f sobre los numeros que le damos.

No todas las funciones tienen funcion inversa. Esto se debe a la definicion de funcion. Para que una relacion sea considerada funcion, para cada elemento del dominio le debe corresponder a lo mas un elemento del contradominio. Si una funcion debe tener funcion inversa, a cada elemento del contradominio le debe corresponder a lo mas un elemento del dominio (por definicion de funcion inversa). En otras palabras, para cada elemento del dominio de f le corresponde un elemento de su contradominio y viceversa.
