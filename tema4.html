Tema 2.4 - Funciones algebraicas: polinomiales y racionales.

Objetivo: Dentro de este tema definiremos las propiedades y principales caracteristicas de las Funciones algebraicas: polinomiales y racionales.

Informacion general

Descripcion general: Las funciones algebraicas son aquellas que se pueden expresar mediante fórmulas algebraicas, es decir, que se pueden escribir en términos de operaciones algebraicas como sumas, restas, multiplicaciones, divisiones, potencias, etc. Dos tipos comunes de funciones algebraicas son las funciones polinomiales y las funciones racionales:

Funciones polinomiales: Son aquellas que se expresan como una suma de términos de la forma ax^n, donde a y n son constantes y x es la variable.

Ejemplo de función polinomial es f(x) = 3x^2 - 2x + 5.

Las funciones polinomiales pueden ser de grado n, donde n es el exponente más alto en la expresión polinomial.

Funciones racionales: Son aquellas que se expresan como el cociente de dos polinomios, es decir, f(x) = P(x)/Q(x), donde P(x) y Q(x) son polinomios y Q(x) no es igual a cero.

Ejemplo de función racional es f(x) = (3x^2 - 2x + 5)/(x + 1).

Las funciones racionales pueden tener puntos de discontinuidad debido a los valores que hacen que el denominador sea igual a cero.

Definicion del tema

Funciones algebraicas: Polinomiales


Las funciones algebraicas polinomiales son una de las funciones más básicas y estudiadas en matemáticas. Están formadas por términos polinomiales, que son expresiones algebraicas que implican sumas, restas y multiplicaciones de potencias de la variable independiente.

Las funciones polinomiales se expresan en la forma general:
f(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_1 x + a_0

Donde:

x es la variable independiente.
a_0, a_1, ..., a_n son los coeficientes reales.
n es un número entero positivo que representa el grado del polinomio. El grado es el exponente más alto de la variable x en la expresión del polinomio.
a_n es el coeficiente del término de mayor grado del polinomio, y se le llama coeficiente líder.

Las funciones polinomiales pueden ser de diferentes grados, según el valor de n en la expresión del polinomio.
Algunos ejemplos de funciones polinomiales son:

f(x) = x^2 + 2x + 1 es una función polinomial de grado 2.
g(x) = x^3 - 4x^2 + 3x + 2 es una función polinomial de grado 3.
h(x) = 4x^4 - x^3 + 3x^2 + x + 2 es una función polinomial de grado 4.

Las funciones polinomiales tienen algunas características notables. La gráfica de una función polinomial es una curva suave y continua, que puede tener ceros o puntos de inflexión.
El dominio de una función polinomial es todo el conjunto de números reales. El rango de una función polinomial también es todo el conjunto de números reales, y puede variar dependiendo del grado del polinomio y de los valores de los coeficientes.

Además, las funciones polinomiales tienen propiedades interesantes, como la simetría par o impar en función del grado y del signo del coeficiente líder. Las funciones polinomiales también
pueden ser sumadas, restadas, multiplicadas y divididas, y se utilizan en muchas áreas de las matemáticas y la física, como la estadística, el cálculo y la teoría de números.

Caracteristicas.


- La gráfica de una función polinomial es una curva suave, sin interrupciones o agujeros.
- El dominio de una función polinomial es todo el conjunto de números reales.
- El rango de una función polinomial también es todo el conjunto de números reales.
- El grado de una función polinomial es el exponente más alto de la variable x. El término con el exponente más alto se llama término líder.
- El signo del término líder determina el comportamiento de la función en los extremos del eje x.
- Las funciones polinomiales pueden ser sumadas, restadas, multiplicadas y divididas.

Ejemplos:

f(x) = x^2 - 3x + 4
g(x) = 4x^3 + 2x^2 - 5x + 1
h(x) = x^4 + 5x^3 - 2x^2 + 7x - 3
p(x) = -2x^5 + 4x^4 - x^3 + 3x^2 - 5x + 2
q(x) = 7x^2 + 6x + 8
r(x) = 2x^6 - x^5 + 4x^4 - 3x^3 + 7x^2 - 2x + 1
s(x) = 3x^2 - 2x + 1/2
t(x) = x^3 - 3x^2 + 2x - 1
u(x) = -x^4 + 6x^3 - 2x^2 + 5x - 1/2
v(x) = 4x^2 + 1/2x - 3

Funciones algebraicas: Racionales


Las funciones algebraicas racionales son aquellas que pueden ser expresadas como una fracción polinómica, donde tanto el numerador como el denominador son polinomios en la variable independiente, y el denominador no es igual a cero. Formalmente, una función algebraica racional se puede escribir de la forma:

f(x) = P(x) / Q(x)

Donde:

x es la variable independiente.
P(x) es un polinomio en x de grado m.
Q(x) es un polinomio en x de grado n, donde n es mayor que cero.
Los coeficientes de los polinomios P(x) y Q(x) son números reales.
Las funciones algebraicas racionales también se llaman funciones fraccionarias o funciones de fracción polinómica. Algunos ejemplos de funciones algebraicas racionales son:

f(x) = (x^2 + 1) / x es una función algebraica racional.
g(x) = (2x^3 - 3x^2 + x) / (x - 1) es una función algebraica racional.
h(x) = (x^4 - 4x^2 + 3) / (x^3 - 2x^2 + x) es una función algebraica racional.
Las funciones algebraicas racionales tienen algunas características notables. El dominio de una función algebraica racional es el conjunto de todos los números reales, excepto aquellos valores de x que hacen que el denominador Q(x) sea cero. Estos valores se llaman puntos de singularidad o puntos de discontinuidad. Los puntos de discontinuidad pueden ser de dos tipos: raíces reales o complejas del denominador y puntos donde el denominador se hace cero pero el numerador no. Además, las funciones algebraicas racionales tienen algunas propiedades interesantes. Pueden ser reducidas a una forma más simple mediante la factorización del numerador y del denominador. También pueden tener asíntotas, que son líneas rectas o curvas que se acercan infinitamente a la gráfica de la función, pero nunca la tocan. Las asíntotas pueden ser verticales, horizontales u oblicuas y se pueden encontrar mediante el análisis de límites. Las funciones algebraicas racionales tienen aplicaciones en muchos campos, incluyendo la física, la ingeniería y las ciencias económicas, y son importantes en el cálculo diferencial e integral.

Caracteristicas.


El dominio de una función algebraica racional está compuesto por todos los valores de x para los cuales el denominador Q(x) no es igual a cero.
Las funciones algebraicas racionales pueden tener puntos de discontinuidad o singularidades, los cuales son valores de x que hacen que el denominador sea igual a cero.
Las funciones algebraicas racionales pueden tener asíntotas verticales, horizontales o oblicuas.
Las funciones algebraicas racionales pueden ser simplificadas mediante la factorización del numerador y denominador.

Ejemplos:



f(x) = (2x + 1) / (x - 3)
g(x) = (x^2 - 4) / (x + 2)
h(x) = (x^3 + 2x^2 + x + 1) / (x^2 - 3x + 2)
i(x) = (3x - 5) / (x^2 + x - 6)
j(x) = (x^2 + 2x + 1) / (x^2 - 1)
k(x) = (4x^3 + 6x^2 + 3x + 1) / (2x^2 - 1)
l(x) = (5x^4 - 3x^2 + 2x) / (x^3 - x)
m(x) = (x^2 - 6x + 9) / (x^2 - 4)
n(x) = (2x^3 - x^2 + 3x + 1) / (x^4 - 4x^2 + 3)
o(x) = (x^5 + x^4 + x^3 + x^2 + x + 1) / (x^3 + x)
